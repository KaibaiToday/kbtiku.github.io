了解流程

【第1题】
下列关于流程的说法中不正确的是( )

A　将串行工序改为并行工序可以节省人员和时间
B　流程设计时要考虑环境这一因素
C　有些时序是可以颠倒的有些时序是不可颠倒的
D　流程是一项活动或一系列连续有规律的事项或行为进行的程序
A B C D
答案：A
解析： 本题考思虻サ牧鞒掏技傲鞒谭治觥＝串行工序改为并行工序,可以节省时间但会增加人员。


【第2题】
下列流程中不合理的是( )



A　制定设计方案的步骤:收集信息二→设计分析一方案构思一方案筛选一方案呈现
B　设计的一般过程:发现与明确问题→制定设计方案→制作模型或原型→测试、评估及优化→编写产品说明书
C　专利申请流程:提交专利申请→受理→初审→专利申请公布→实质审查一授权
D　金属材料的划线步骤:划基准线→划加工尺寸线一划轮廓线→冲眼
A B C D
答案：A
解析： 本题考思虻サ牧鞒掏技傲鞒谭治觥Ｖ贫ㄉ杓品桨附锥斡ο确桨赋氏衷俜桨干秆


【第3题】
某毛衫生产公司通过校企合作，开发了新型电脑横机（电脑横切机），优化了毛衫生产工艺流程，缩短了生产周期，提高了毛衫的质量。下列关于该工艺流程优化的说法中，不正确的是（ ）

A　该工艺流程的优化属于技术优化
B　新型电脑横机（电脑横切机）的应用是该工艺流程优化的主要条件
C　对毛衫生产过程的研究是该工艺流程优化的基础
D　该工艺流程的优化实现了单一目标
A B C D
答案：D
解析： 考查对流程优化的分析。流程的优化与材料、设备、工艺等有密切的关系。某毛衫生产企业通过校企合作，对毛衫生产过程的研究是该流程优化的基础，故对毛衫生产过程的研究是该工艺流程优化的基础，正确:优化了毛衫生产工艺流程，缩短了生产周期，提高了毛衫的质量，实现了多个目标, 而非单一目标，符合题意。该工艺流程优化使用了新型电脑横机（电脑横切机），应用了新技术，属于技术优化，正确;新型电脑横机的应用是该工艺流程优化的条件，正确。

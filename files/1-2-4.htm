【第1题】
下图为某零件的三视图，其对应的轴测图是



A B C D
答案：A
根据左视图可以看出，该零件由两部分连接而且两部分的宽度相近，因而排除了选项C和选项D；根据俯视图可以看出，该零件有一部分是圆柱状的，排除了选项B。选项A是该零件的轴测图。


【第2题】
如图所示是某一形体的轴测图，其正确的主视图是( )





A B C D
答案：B
解析：三视图的3个特性：积聚性、收缩性、真实性。


【第3题】

A B C D
答案：C
解析：实线和虚线的表示。


【第4题】
如图所示的立体图及其主视方向，其对应的主视图是（　　）





A B C D
答案：B
解析：水平方向平行光线投影。


【第5题】
用来表达隐藏在表层后面、内部、下面的不可见结构的技术图样是（）。





A　剖视图
B　草图
C　机械加工图
D　电子电路图
A B C D
答案：A
解析：剖视图是用来表达隐藏在表层后面、内部、下面的不可见结构的技术图样。
